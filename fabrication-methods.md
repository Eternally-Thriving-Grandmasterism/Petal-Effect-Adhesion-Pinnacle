# Petal Effect Adhesion Fabrication Methods Lattice

Recommended Scalable Path: Multi-Level Nanoimprint Lithography (NIL) + Anisotropic Texturing

1. Master Mold Preparation
   - Si/Ni shim: e-beam/LIL for dense 5–20 μm pillars + 100–500 nm folds/creases

2. Resin Coating & Imprint
   - R2R UV resin on PET/PC → multi-level NIL (micro first → nano overlay)

3. Demold & Surface Modification
   - Separate → optional plasma activation + hydrophobic coat (fluorosilane / wax)

4. Anisotropy Tuning
   - Tilted etching or directional deposition for directional pinning

Alternative Paths:
- Colloidal lithography + sol-gel (spheres for pillars → folds via shrinkage)
- Femtosecond laser texturing (direct on polymer/metal)

Valence Check:
- Dry processes preferred
- Recyclable masters
- Low-waste → eternal mercy
